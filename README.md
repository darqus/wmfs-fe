## Run Scripts

### Install the dependencies

```sh
$ yarn
# or
$ npm i
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)

```sh
$ yarn quasar dev
# or
$ npx quasar dev
```

### Linting errors with ESLint

```sh
$ yarn lint-fix
# or
$ npm run lint-fix
```

### Compiles and minifies for production

```sh
$ yarn quasar build
# or
$ npx quasar build
```

## Tests

### With UI

```sh
$ yarn vitest --ui
# or
$ npx vitest --ui
```

### With CLI

```sh
$ yarn vitest

$ npx vitest
```

### For CI

